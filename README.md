Thank you for downloading Universal Converter Pro!

This is a lightweight, all-in-one tool designed to convert media files 
(Images, Videos, Audio) and compress large documents without needing 
expensive software.



------------------------------------------------------------------------
                           KEY FEATURES
------------------------------------------------------------------------

1. UNIVERSAL CONVERTER
   - Images: Convert between JPG, PNG, WEBP, BMP, and HEIC (iPhone).
   - Video: Convert MP4, MKV, AVI, MOV, FLV.
   - Audio Extraction: Turn any video file into MP3 or WAV audio.

2. SMART COMPRESSOR
   - Images: Reduce file size by up to 90% using "Extreme Compression".
   - PDFs: Optimize PDF streams to shrink document size for emails.
   - 3 Levels: Choose between "High Quality", "Balanced", or "Extreme".

3. MODERN INTERFACE
   - Dark Mode design using CustomTkinter.
   - Fully responsive window (resizeable).
   - "Save As" dialog allows you to rename files before processing.

------------------------------------------------------------------------
                        HOW TO USE
------------------------------------------------------------------------

A. TO CONVERT FILES:
   1. Open the "Converter" tab.
   2. Click "Browse" to select your file.
   3. Choose your desired output format from the list.
   4. Click "Convert File" and choose where to save it.

B. TO COMPRESS FILES:
   1. Open the "Compressor" tab.
   2. Select a large Image or PDF file.
   3. Choose your Compression Level:
      - Medium: Best balance (Recommended).
      - Extreme: Maximum size reduction (Quality will be lower).
      - Low: Keeps high quality, only removes metadata.
   4. Click "Compress File".

------------------------------------------------------------------------
                        TROUBLESHOOTING
------------------------------------------------------------------------

- "Error: No audio track":
  This happens if you try to convert a silent video to MP3. Please ensure 
  the source video has sound.

- "Extreme Compression looks blurry":
  This is normal. Extreme mode resizes images to 1024px and reduces 
  colors to save maximum space. Use "Medium" for better quality.

- App doesn't open?
  This is a portable application. No installation is required, but it 
  is designed for Windows 10/11.
